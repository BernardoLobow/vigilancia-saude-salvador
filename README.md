# 🏥 Vigilância em Saúde - Engenho Velho (Salvador/BA)

Este é um portal de transparência e participação popular focado na saúde pública. O sistema permite que cidadãos acompanhem dados sobre doenças e reportem incidentes de descarte incorreto de lixo, auxiliando no combate a focos de doenças como a Dengue e Leptospirose.

## 🚀 Funcionalidades

- **Painel de Informações**: Listagem dinâmica de doenças com sintomas e métodos de prevenção.
- **Notificação Epidemiológica**: Formulário para registro de casos suspeitos.
- **Denúncia de Descarte de Lixo**: Sistema com upload de imagens para reportar focos de poluição urbana.

## 🛠️ Tecnologias Utilizadas

- **Frontend**: HTML5, CSS3 (Customizado), Bootstrap 5 e JavaScript (Vanilla).
- **Backend**: PHP 8.x.
- **Banco de Dados**: SQLite para persistência leve e rápida.
- **Comunicação**: Fetch API para submissões assíncronas (AJAX).
